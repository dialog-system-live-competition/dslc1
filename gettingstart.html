How to use Telegram

Telegramボットの作成方法

  1. 1. Telegramでユーザ名 @BotFather を検索し，+Joinでフレンドに追加する
  2. 2. @BotFatherに /newbot と送信
  3. 3. ボット名を聞かれるので入力 (日本語も使用可)　※エントリフォームに入力したボット名と同じにしてください．
  4. 4. ボットのユーザ名を聞かれるので入力．半角英数字で最後はbotで終わる必要がある．また，他に同じユーザ名のボットがある場合は登録できない．
  5. 5. アクセストークンが発行される．プログラム上でこのアクセストークンを指定する必要がある．
